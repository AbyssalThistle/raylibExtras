Copy the raylib source code folder into the root folder, the same place REPLACE_THIS_WITH_RAYLIB_SOURCE is.
Open RLTilesTest folder
Open all files in a text editor and replace all instances of RLTilesTest with your project name.
Rename all SAMPLE_PROEJCT.* files to your project name, be sure to keep the extensions
Go back to the root folder
Rename the RLTilesTest folder to your project name.
Open RLTilesTest.sln in a text editor and replace all instances of RLTilesTest with your project name.
Rename RLTilesTest.sln to your project name.sln

Open the solution file in visual studio 2019 and you are good to go.
